# Linux System Security

## Description
Ce dépôt présente des travaux pratiques réalisés dans le cadre de ma formation
en Sécurité des systèmes et réseaux informatiques.

## Objectifs
- Comprendre l’administration d’un système Linux
- Appliquer les bonnes pratiques de sécurité système
- Sécuriser les accès et les ressources

## Compétences abordées
- Gestion des utilisateurs et groupes
- Permissions et droits d’accès
- Sécurisation du service SSH
- Mise en place d’un firewall (UFW)
- Bonnes pratiques de sécurité Linux

## Contexte
Projets académiques et démonstratifs réalisés à des fins
d’apprentissage et de renforcement des compétences professionnelles.
